# 面向高采样率准周期信号的平滑主波形预测研究方案

## 0. 文档定位

本文档用于指导后续论文研究、代码实现、实验设计和结果整理。当前研究方向已经从“单一燃烧不稳定预测”重构为“面向高采样率准周期信号的未来平滑主波形预测框架”。

本研究的核心目标不是预测原始高频信号的每一个未来采样点，也不是只做燃烧失稳风险分类，而是建立一个可配置、可迁移的准周期信号预测框架：输入历史原始信号，输出未来平滑主波形，并通过峰值、峰间隔、包络和频谱结构指标评价模型是否真正保留准周期结构。

## 2. 研究背景与动机

### 2.1 原始问题

最初的研究对象是不稳定燃烧压力信号预测。该类数据具有以下特点：

1. 采样率极高，例如 0.5 秒内可能包含约 50 万个采样点；
2. 信号存在明显准周期结构，表现为连续波峰、波谷和包络变化；
3. 原始高频细节中包含大量噪声、局部扰动、相位漂移和可能不可稳定预测的混沌成分；
4. 直接长步预测原始 raw waveform 很容易在 100 多步后失真，而这 100 多步对应的真实物理时间又非常短；
5. 若只做燃烧不稳定数据集，公开数据稀缺，论文风险较高。

因此，原来的“长时逐点原始波形预测”并不是最稳妥的论文主线。

### 2.2 新的研究转向

观察燃烧数据后可以发现，原始信号虽然高频复杂，但其平滑后的主波形仍然保留了重要的周期结构：

- 波峰位置；
- 波谷位置；
- 峰值幅度变化；
- 峰间隔变化；
- 包络上升或下降趋势；
- 主频或低频结构变化。

因此，本研究将预测目标从：

> 预测未来原始高频信号的每一个采样点

重构为：

> 预测未来平滑主波形，并保持其中的准周期结构

这使得任务既保留“波形预测”的直观性，又避免被不可稳定预测的高频残差拖垮。

## 3. 核心研究问题

本研究围绕以下三个核心问题展开。

### RQ1：高采样率准周期信号是否适合直接做原始逐点长时预测？

初步判断：不适合直接作为论文主线。

原因是原始高频信号中存在大量局部噪声、高频残差和相位敏感成分。长时逐点预测时，即使模型学到了主周期结构，也可能因为微小相位偏移导致点对点误差迅速增大。因此，直接预测 raw future 容易低估模型对主结构的预测能力。

### RQ2：预测未来平滑主波形是否能保留原始信号中最重要的可预测结构？

初步判断：可以。

平滑主波形保留了准周期信号中最具有稳定性和物理意义的部分，包括峰谷、包络、周期和主频变化。你的燃烧实验图已经说明，模型能够较好预测 smoothed ground truth 的未来形态。

### RQ3：一个统一框架是否可以扩展到其他准周期数据集？

初步判断：可以，但边界必须清楚。

可靠主张是：

> 同一个模型框架可以在不同准周期数据集上分别训练或少样本微调，用于预测未来平滑主波形。

不应主张：

> 一个训练好的单一权重可以零样本预测所有准周期数据。

跨数据集权重迁移可以作为增强实验，但不能作为论文唯一支柱。

## 4. 研究对象与任务定义

### 4.1 信号分解假设

对高采样率准周期信号，可以近似看作：

\[
x_{raw}(t)=x_{smooth}(t)+r(t)
\]

其中：

- \(x_{raw}(t)\)：原始高频信号；
- \(x_{smooth}(t)\)：平滑主波形，包含可预测的周期、包络、趋势和低频结构；
- \(r(t)\)：高频残差、噪声、局部毛刺、随机扰动或不可稳定预测成分。

本研究主要预测 \(x_{smooth}(t)\)，而不是强行预测所有 \(r(t)\)。

### 4.2 统一任务定义

对于任意准周期数据集，统一构建以下监督学习任务：

\[
X = x_{raw}[t-L:t]
\]

\[
Y = x_{smooth}[t:t+H]
\]

即：

> 输入过去一段原始信号，预测未来一段平滑主波形。

其中：

- \(L\)：历史窗口长度；
- \(H\)：未来预测窗口长度；
- \(x_{smooth}\)：由 moving average、Savitzky-Golay、low-pass filter、Hilbert envelope 等方法生成，具体取决于数据类型。

### 4.3 非通用任务说明

风险状态不是所有数据集的通用输出。

燃烧数据可能存在 safe / warning / unstable 等风险状态；轴承数据可能存在故障趋势；ECG 可能存在异常节律。但这些都是数据集特定任务。

因此本研究中：

- 通用主任务：future smooth waveform forecasting；
- 可选任务：dataset-specific risk / event / anomaly head。

## 5. 研究贡献设计

本文不能只写成“把一个现有模型套到不同数据集上”，否则创新会偏弱。更合理的定位是提出一个**通用准周期预测框架**，并在框架内放入 1～2 个真正可实现的方法创新。

本文可形成以下四个主要贡献。

### Contribution 1：任务重定义

提出面向高采样率准周期信号的未来平滑主波形预测任务，将原始逐点长时预测重构为更稳定、更具有结构意义的 smooth waveform forecasting。

### Contribution 2：统一准周期预测框架

提出 **QPF-SmoothPEC** 框架，即 Quasi-Periodic Forecasting with Smooth Peak–Envelope–Cycle Modeling。该框架把不同来源的准周期信号统一成同一个任务：

`raw history -> future smooth main waveform`

并联合利用 raw signal、smooth/envelope view 和 cycle tokens，预测未来平滑主波形。

这个贡献强调的是：同一套框架可以适配燃烧压力、轴承振动、生理 PPG/呼吸信号和合成准周期信号，而不是为某一个数据集写死模型。

### Contribution 3：方法创新一：准周期结构保持约束

设计峰值、峰间隔、包络和频谱结构指标，用于评价和约束预测波形是否真正保留准周期结构，而不是只降低点误差。

具体创新点可以写成：

> **Quasi-periodic structure-preserving loss**：在普通波形误差之外，引入 derivative、spectral、peak/envelope 等结构约束，使模型预测出来的平滑主波形不仅 MAE 低，而且峰谷位置、包络趋势和主频结构更接近真实未来。

这是第一处方法创新。

### Contribution 4：方法创新二：周期归一化的通用数据接口与可迁移框架

提出一个面向不同采样率、不同周期长度、不同物理来源信号的统一数据接口：先从原始信号中估计局部周期或主频，再按“历史若干周期 -> 未来若干周期”的方式构造样本，并输出统一长度的 future smooth waveform。

这个贡献可以写成：

> **Cycle-normalized forecasting protocol**：不直接用固定采样点长度比较不同数据，而是用历史周期数、未来周期数和重采样后的平滑主波形作为统一预测单位，从而让燃烧、PPG/呼吸、轴承振动等准周期数据能进入同一个框架。

这是第二处方法创新。

在燃烧压力、生理准周期信号、工业轴承振动和合成准周期信号上验证该框架的通用性，并进一步探索预训练—微调对少样本目标数据的帮助。

## 6. 数据集规划

### 6.1 第一阶段：已经验证的燃烧数据

燃烧压力数据是本研究的核心动机和第一成功案例。

任务形式：

\[
raw\ pressure\ history \rightarrow future\ smooth\ pressure
\]

重点验证：

- 模型能否预测未来平滑波形；
- 预测结果是否保留波峰、波谷和包络趋势；
- 原始 raw future prediction 与 smooth future prediction 的差异；
- 不同平滑窗口下结果是否稳定。

### 6.2 第二阶段：CWRU 轴承振动数据

CWRU 用于验证框架是否能处理工业高频振动信号。

第一轮只使用 4 个文件：

1. Normal_0.mat；
2. IR007_0.mat；
3. B007_0.mat；
4. OR007@6_0.mat。

任务形式：

\[
raw\ vibration\ history \rightarrow future\ smooth\ envelope
\]

注意：CWRU 第一轮不做故障分类，不预测 raw vibration future，而是预测未来平滑包络。

### 6.3 第三阶段：BIDMC / PPG / Respiration 数据

BIDMC 用于验证框架是否能迁移到生理准周期信号。

优先信号：

- respiration；
- PPG。

任务形式：

\[
raw\ respiration/PPG\ history \rightarrow future\ smooth\ waveform
\]

### 6.4 第四阶段：合成准周期数据

合成数据用于机制验证。

生成形式：

\[
x(t)=A(t)\sin(\phi(t))+noise(t)+burst(t)
\]

可控因素：

- 幅值调制；
- 频率漂移；
- 多频叠加；
- 突发大峰；
- 噪声强度；
- 缺失峰。

合成数据的作用不是刷分，而是验证模型是否真的适用于可控准周期变化。

## 7. 模型框架：QPF-SmoothPEC

本研究建议把模型整体命名为 **QPF-SmoothPEC**，全称可以是：

**Quasi-Periodic Forecasting with Smooth Peak–Envelope–Cycle Modeling**

其中 QPF 表示这是一个通用准周期预测框架，Smooth-PEC 表示该框架内部使用平滑主波形、峰值、包络和周期结构共同建模。

### 7.1 总体结构

QPF-SmoothPEC 的基本结构如下：

```text
historical raw signal
    ↓
cycle-aware preprocessing: smoothing / envelope / peak extraction / period estimation
    ↓
cycle-normalized window construction
    ↓
multi-view encoders
    ├── raw branch
    ├── smooth or envelope branch
    └── cycle-token branch
    ↓
quasi-periodic fusion module
    ↓
decoder
    ↓
future smooth main waveform
```

### 7.2 Raw Branch

输入历史原始信号，用于保留局部形态、高频调制和周期内部细节。

第一版可使用：

- 1D-CNN；
- TCN；
- lightweight Conv1D encoder。

### 7.3 Smooth / Envelope Branch

输入历史平滑波形或包络，用于学习慢变量趋势和低频主结构。

适用于：

- 燃烧 pressure smooth；
- CWRU smooth envelope；
- PPG smooth waveform；
- respiration smooth waveform。

### 7.4 Cycle-token Branch

从历史信号中提取周期 token：

\[
C_i=[a_i,\Delta t_i,width_i,prominence_i,energy_i]
\]

其中：

- \(a_i\)：峰值幅度；
- \(\Delta t_i\)：峰间隔；
- \(width_i\)：峰宽；
- \(prominence_i\)：峰突出度；
- \(energy_i\)：周期能量。

该分支用于帮助模型显式感知周期结构。

### 7.5 推荐的 1～2 个模型创新

为了让论文不是简单套模型，建议把模型创新集中在两个可实现点上。

#### 创新点 1：Cycle-normalized forecasting interface

不同数据集采样率差异很大，直接固定点数建模会让任务不统一。这里引入周期归一化接口：

1. 对每条信号估计局部主频或峰间隔；
2. 用“过去 N 个周期”作为历史窗口；
3. 用“未来 M 个周期”作为预测窗口；
4. 将 future smooth waveform 重采样到统一长度，例如 512 或 1024 点；
5. 模型输入输出接口保持一致。

这样模型看到的是准周期结构，而不是某一个数据集特有的采样率。

#### 创新点 2：Structure-preserving smooth waveform loss

主输出仍然是 future smooth waveform，但训练时加入结构保持约束：

`L = L_wave + a * L_derivative + b * L_spectral + c * L_peak_or_envelope`

其中：

- `L_wave`：整体平滑波形误差；
- `L_derivative`：趋势和上升下降形态；
- `L_spectral`：主频和频谱结构；
- `L_peak_or_envelope`：峰值与包络结构。

这能让预测结果不仅像一条平滑曲线，而且保留准周期信号真正重要的峰谷、包络和频率特征。

### 7.6 输出

主输出：

\[
\hat{x}_{smooth}[t:t+H]
\]

即未来平滑主波形。

可选输出：

- 未来峰值；
- 未来峰间隔；
- 未来包络趋势；
- 数据集特定风险或事件状态。

## 8. 损失函数设计

### 8.1 主损失

主损失是波形误差：

\[
L_{wave}=MAE(\hat{x}_{smooth},x_{smooth})
\]

或：

\[
L_{wave}=MSE(\hat{x}_{smooth},x_{smooth})
\]

第一版建议使用 MAE，因为对局部异常点更稳。

### 8.2 趋势损失

为了保持上升下降趋势，可以加入一阶差分损失：

\[
L_{derivative}=MAE(\Delta \hat{x}_{smooth},\Delta x_{smooth})
\]

### 8.3 频谱损失

为了保持主频和频域结构，可以加入频域损失：

\[
L_{spectral}=MAE(|FFT(\hat{x})|,|FFT(x)|)
\]

### 8.4 峰值与周期结构损失

可加入：

- peak amplitude loss；
- peak timing loss；
- interval loss；
- envelope loss。

第一版不必全部加入，建议按阶段添加。

### 8.5 总损失

完整形式：

\[
L=L_{wave}+\lambda_1L_{derivative}+\lambda_2L_{spectral}+\lambda_3L_{peak}+\lambda_4L_{interval}+\lambda_5L_{envelope}
\]

第一阶段可先使用：

\[
L=L_{wave}
\]

第二阶段再逐步加入结构损失。

## 9. 评价指标

### 9.1 波形整体误差

- MAE；
- RMSE；
- nMAE；
- nRMSE；
- Pearson correlation；
- DTW distance。

### 9.2 周期结构误差

- peak amplitude error；
- peak timing error；
- peak interval error；
- future max error；
- large peak recall。

### 9.3 包络与频谱误差

- envelope MAE；
- envelope slope error；
- dominant frequency error；
- spectral similarity；
- band-energy error。

### 9.4 统计规范

正式论文实验中应报告：

- 至少 5 个随机种子；
- mean ± std；
- 明确的 train / validation / test 划分；
- train-only 标准化；
- 不随机打乱滑窗后再划分。

## 10. 实验路线

### 10.1 阶段 A：燃烧单数据集确认

目标：证明 raw history → smooth future 是稳定可复现的。

实验：

1. raw future prediction vs smooth future prediction；
2. fixed horizon prediction；
3. horizon extension；
4. last-window repeat baseline；
5. TCN baseline；
6. 当前模型预测图。

通过标准：

- 模型优于 last-window repeat；
- 预测图能跟住未来平滑波形；
- correlation 明显为正；
- 峰谷结构不完全错位。

### 10.2 阶段 B：CWRU 最小验证

目标：快速验证其他高频物理数据是否也能做 future smooth waveform prediction。

当前任务：

\[
raw\ vibration \rightarrow future\ smooth\ envelope
\]

最小实验：

1. 检查 .mat 变量；
2. 提取 DE_time；
3. Hilbert envelope + moving average；
4. 构造窗口；
5. last-window repeat baseline；
6. 轻量 TCN 训练；
7. 输出 prediction vs ground truth smooth envelope 图。

通过标准：

- smooth envelope 不是一条直线；
- TCN 能超过 last-window repeat；
- 预测图能跟住未来 envelope 趋势；
- 值得后续加入完整实验。

### 10.3 阶段 C：BIDMC / PPG / Respiration 验证

目标：验证框架能处理生理准周期信号。

任务：

\[
raw\ PPG/respiration \rightarrow future\ smooth\ waveform
\]

先做最小验证，不急于复杂 baseline。

### 10.4 阶段 D：统一框架与多数据集实验

当燃烧、CWRU、BIDMC 都能跑通后，开始整理成统一框架。

实验包括：

- 多数据集分别训练；
- 同一模型代码和配置接口；
- 每个数据集输出相同格式结果；
- 统一评价指标。

### 10.5 阶段 E：迁移学习探索

该阶段是增强实验，不是主线前提。

实验：

1. source datasets pretraining；
2. target dataset few-shot fine-tuning；
3. scratch vs pretrain comparison；
4. 5%、10%、20%、50% 目标数据量实验。

可靠结论应写为：

> 跨准周期预训练在部分少样本场景下可能提升平滑波形预测效果，但是否稳定有效需要依赖源域和目标域相似性。

不能写成：

> 一个权重可以直接预测所有准周期数据。

## 13. 风险与止损方案

### 13.1 如果 CWRU 预测不好

可能原因：

- smooth envelope 太平；
- 轴承信号不适合当前窗口；
- pred_len 太长；
- 平滑方法不合适；
- raw vibration 到 envelope 的映射太难。

处理：

- 缩短 pred_len；
- 改用 RMS trend；
- 改用 lower frequency envelope；
- 只把 CWRU 放到附录或放弃。

### 13.2 如果 BIDMC 预测很好但 CWRU 不好

这并不影响主线。论文可以先覆盖燃烧 + 生理 + synthetic，工业振动作为探索性实验。

### 13.3 如果迁移学习无提升

不把迁移写成主贡献，只保留为探索实验。主贡献仍然是统一任务定义、Smooth-PEC 框架和多数据集分别训练验证。

### 13.4 如果辅助结构损失没有降低 MAE

不写“所有指标全面提升”，而写“结构损失提升峰值、包络或频谱保持能力”。

## 14. 预期论文结论

如果实验顺利，论文可以形成如下结论：

1. 高采样率准周期信号的原始逐点长时预测困难，平滑主波形预测是更稳定的任务定义；
2. Smooth-PEC 能够在燃烧等数据上预测未来平滑波形，并保持主要峰谷和包络结构；
3. 峰值、峰间隔、包络和频谱指标能够更全面评价准周期预测质量；
4. 同一框架可以扩展到多种准周期数据集，说明该方法不是燃烧数据专属；
5. 跨数据集预训练可能在少样本目标数据中提供帮助，但需要谨慎评价。

## 16. 最终研究定位

本研究的最终定位应写为：

> 以燃烧不稳定预测为动机，本文提出一种面向高采样率准周期信号的通用未来平滑主波形预测框架 QPF-SmoothPEC。该框架不追求不可稳定预测的原始高频细节，而是预测信号中更具稳定性和物理意义的平滑主波形。方法上，本文引入周期归一化预测接口与准周期结构保持损失，使不同采样率、不同周期长度、不同物理来源的准周期信号能够进入统一预测框架，并通过峰值、包络、周期和频谱结构指标评价预测结果的结构保持能力。进一步地，本文在燃烧、生理、工业振动和合成准周期数据上验证该框架的可配置性和迁移潜力。

这一定义既保留了你现有燃烧预测图的优势，又避免将论文局限在单一燃烧数据集或不可控的 raw waveform 长时预测上。
